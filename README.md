Quick Cheat Sheet:
100 = Continue
200 = OK
201 = Created
204 = No Content
301 = Moved Permanently
302 = Found (Temporary Redirect)
400 = Bad Request
401 = Unauthorized
403 = Forbidden
404 = Not Found
409 = Conflict
500 = Internal Server Error
502 = Bad Gateway
503 = Service Unavailable
504 = Gateway Timeout
